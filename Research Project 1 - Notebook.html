MAUVE Genome Alignments

  • A MAUVE multiple genome alignment (version 2.4.0) between the two published genomes for the naked mole rat was created in a Linux terminal (Bradford University SSH server)
            Note: I downloaded 'Putty' (and pscp - "ps copy") in order to SSH into Dr Andrew Tedder's university server and copy the mauve .xmfa file onto my home PC.

Below is the command typed into the terminal to generate the mauve alignment :

progressiveMauve --output=H_glaber.xmfa /home/bilalm/H_glaber_genomes/GCA_000230445.1_HetGla_1.0_genomic.fna /home/bilalm/H_glaber_genomes/ GCA_000247695.1_HetGla_female_1.0_genomic.fna
 
The alignment consisting of both NMR assemblies was created using the 'progressiveMauve' algorithm which produced three output files :

(1) .XMFA file - containing the Mauve genome alignments
(2) .backbone file - containing records of conserved regions
(3) .bbcols file - containing records of (predicted) alignment positions.

Note: It IS possible for the .XMFA file to be graphically visualized using the Mauve-Viewer (version 20150226). This work is still ongoing. 

To view the .XMFA file in the Linux terminal type :

bilalm@Milou:~/H_glaber_genomes/mauve_alignment$ less h_glaber.xmfa

 

Reference:
NNNNNN - Gaps (Ambiguous base calls) 
(uppercase) ATCGATCG - confident base calls
(lowercase) atcgatcg - less confident base calls.



QUAST - Genome Assembly Assessment Tool

  • QUAST is a quality assessment tool for evaluating/comparing genome assemblies.
  • QUAST tool in Galaxy (Version 5.0.2) used to generate a number of summary statistics (in numerical/graphical format) in order to gauge an assembly quality for the naked mole-rat genomes. 

Step 1 - Download both genomes (HetGla_1.0 & HetGla_female_1.0) from the NCBI website under accession numbers ‘GCF_000230445.1’ & ‘GCF_000247695.1’ in FASTA/.fna format. 

Step 2 - A decompression step was required as the newly downloaded .fna files were in a compressed .gz format. To compress the genome sequence files use: 

$ gunzip GCA_000230445.1_HetGla_1.0_genomic.fna.gz
$ gunzip GCF_000247695.1_HetGla_female_1.0_genomic.fna.gz


Step 4 - Import the decompressed FASTA files into Galaxy. Wait for upload, then select the Quast Genome assembly Quality (Galaxy Version 5.0.2+galaxy1) tool. 

Step 5 - Wait for QUAST to run. Quast generates (1) tabular report (2) HTML report (3) PDFG report ($) Log .txt file.  

Note: Results are not included in this notebook. To see the results of QUAST for NMR genomes go to :  https://github.com/mbilal1995/Research-Project_1
